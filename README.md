# Atom

Atom is a Rust-first mobile application framework that scaffolds Bazel-native apps and generates iOS
and Android host code from metadata.

If you want to go from zero to a running app, start with
[docs/getting-started.md](docs/getting-started.md).

For deeper repository docs, see [docs/README.md](docs/README.md). For architecture details, see
[docs/architecture.md](docs/architecture.md). For the current specification, see [SPEC.md](SPEC.md).
